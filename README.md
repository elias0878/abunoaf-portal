# Abu Noaf Digital Portal

منصة أبو نواف الرقمية - منصة وطنية سعودية متخصصة في الأمن السيبراني

## المميزات
- 10 أقسام رئيسية
- واجهة عربية احترافية
- تصميم يتوافق مع المعايير الحكومية السعودية
- دعم كامل للغة العربية وRTL

## التقنيات المستخدمة
- React + TypeScript
- Tailwind CSS
- shadcn/ui
